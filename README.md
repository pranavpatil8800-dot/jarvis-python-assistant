# 🤖 Jarvis Python Assistant

A voice-controlled virtual assistant inspired by Iron Man’s Jarvis.
This project allows users to perform tasks using voice commands without typing.

---

## 🚀 Features

### 🎤 Voice Recognition

* Fully voice-controlled system
* No need to type commands
* Just speak and the assistant performs tasks

---

### 💬 Smart Responses

* Responds to greetings like "Hello"
* Interactive communication with the user

---

### 🖥️ Application Control

* Open any application using voice commands
* Example: "Open Chrome", "Open Notepad"

---

### 📩 WhatsApp Automation

* Send WhatsApp messages using voice
* User provides:

  * Phone number
  * Message
  * command: send whatsapp "Whats app No."
* Automatically stores message data in **PyWhatKit_DB**:

  * Message content
  * Receiver number
  * Date & time

---

### 📝 Task Management System

* Add tasks to a text file using voice

  * Command: "Add task"
* Tasks are saved in: **task file.txt**

---

### 📖 Task Reading

* Read saved tasks using voice

  * Command: "Speak list"

---

### 📋 Task Display

* Show all saved tasks on screen

  * Command: "Show work"

---
